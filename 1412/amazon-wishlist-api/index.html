Amazon Wish List API for Scala

By William Narmontas, Polyglot Software Engineer

Sample project showing how to extract items from your Amazon Wishlist, turn them into JSON, serve them via Spray, to an AngularJS interface, all looked after by SBT 0.13.7 and its native packager plugin, under Scala 2.11.4
